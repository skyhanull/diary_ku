# 파운데이션

## 색상

### 브랜드 / 핵심 색상
- `Cedar` `#8C6A5D`
  - 핵심 브랜드 컬러
  - 주요 CTA, 브랜드 포인트, 강조 액션에 사용
- `Vellum` `#FDF8F5`
  - 메인 배경
  - 종이 같은 오프 화이트 톤

### 보조 / 표면 색상
- `Raw Umber` `#6B4C40`
  - 주요 텍스트, 대비가 필요한 핵심 UI
- `Oatmeal` `#F8F3EF`
  - 보조 배경, 카드, 약한 구획 분리
- `Stone Gray` `#34322F`
  - 아이콘, 보조 텍스트, 차분한 라벨

### 상태 색상
- `Sage` `#8C9B8C`
  - 성공, 완료, 안심 피드백
- `Dusty Rose` `#B88E8E`
  - 에러, 삭제, 파괴적 액션

## 표면 계층
- 기본 캔버스: `Vellum`
- 보조 영역: `Oatmeal`
- 떠 있는 패널 또는 강조 카드: `Vellum` 또는 더 밝은 카드색 + 얕은 경계선

## 경계 원칙
- 강한 1px 구분선보다 톤 차이로 구역을 나눈다.
- 다만 캘린더, 에디터 패널, 폼에서는 낮은 대비의 얇은 선을 허용한다.
- 무거운 섹션 박스보다 배경 레이어링을 우선한다.

## 타이포그래피

### 기본 헤드라인 폰트
- `Epilogue`
- 영문 헤드라인, 브랜드성 있는 큰 제목, 강조 문구에 사용
- Medium~Bold 권장
- 자간은 약 `-0.02em`

### 기본 UI / 본문 폰트
- `Pretendard`
- 한글 UI, 본문, 입력창, 라벨, 일기 내용에 기본 사용
- 본문 줄간격은 `1.6`
- 장식성보다 가독성을 우선

### 보조 에디토리얼 폰트
- `Noto Serif KR`
- 날짜 스탬프, 짧은 인용문, 감성 강조 텍스트에 제한적으로 사용
- 기본 UI 폰트로는 사용하지 않음

## 모서리 규칙
- 기본 radius는 `8px`
- 버튼, 카드, 입력창은 부드럽지만 구조적인 인상을 유지한다
- 특별한 강조 요소만 더 큰 radius를 허용한다

## 그림자 규칙
- 대부분의 표면은 그림자 없이 둔다
- 떠 있는 패널, 모달, 플로팅 요소만 매우 약한 ambient shadow 사용
- 권장 그림자:

```css
box-shadow: 0 12px 32px rgba(52, 50, 47, 0.04);
```

## 질감 규칙
- 큰 배경에는 아주 약한 노이즈나 종이 질감을 사용할 수 있다
- 질감은 장식이 아니라 디지털 평면감을 줄이는 보조 수단이어야 한다
- 텍스트 가독성을 해칠 정도의 질감은 금지한다
