# 브랜드 방향

## 크리에이티브 노스스타
DearMe의 디자인 시스템은 `디지털 헤리티지 저널` 감성을 목표로 한다.
빠르고 차가운 생산성 툴이 아니라, 오래 간직하고 싶은 다이어리와 스크랩북의 중간 지점을 지향한다.

## 핵심 무드
- 따뜻함
- 종이 같은 질감
- 조용한 프리미엄
- 개인적이고 사적인 분위기
- 아기자기하지만 과하지 않은 디테일

## 비주얼 인상
- 부드러운 오프 화이트 배경
- 나무와 종이를 떠올리게 하는 브라운 계열 포인트
- 과한 그림자보다 톤 차이와 얇은 경계선
- 둥글지만 지나치게 말랑하지 않은 형태

## 피해야 하는 인상
- 기업용 SaaS 대시보드 느낌
- 과한 보라색 중심 테마
- 차갑고 유리 같은 미래형 UI
- 정보 밀도가 높은 관리자 화면
- 너무 장난감 같거나 유치한 키즈 앱 느낌

## DearMe에서의 해석
이 시스템은 감성만 강조하는 것이 목적이 아니다.
실제 사용자는 날짜를 고르고, 기록을 열고, 꾸미고, 저장해야 한다.
따라서 감성은 구조를 덮는 장식이 아니라 사용 흐름을 부드럽게 만드는 배경이어야 한다.
